# F* Those Guys! 🚩

**The "Mission-First" Blocklist for Network Safety.**

Most blocklists focus on blocking ads or malware. **F* Those Guys!** is different—it focuses on **intent**. This list targets platforms that have demonstrated a clear disregard for user safety, specifically those that facilitate predatory behavior, host unmoderated "stranger" interactions, or profit from the exploitation of minors.

### Why this list exists:
* **Zero Accountability:** We block platforms like Roblox that have consistently failed to protect children from documented grooming and predatory behavior.
* **Predatory by Design:** We target "Stranger Chat," "Anonymous Q&A," and "AI Companion" sites that serve as digital playgrounds for predators.
* **Unregulated Exploitation:** We block data brokers and unregulated gambling sites that target youth and vulnerable users.

---

### 🚀 How to Use

#### 1. Copy the Raw Link
Copy and paste this URL into your DNS blocker:
`https://raw.githubusercontent.com/ebarber972/f-those-guys/main/f-those-guys-blocklist.txt`

#### 2. Add to Pi-hole
* Navigate to **Group Management > Adlists**.
* Paste the URL and add the comment: `F* Those Guys!`.
* Go to **Tools > Update Gravity** to apply.

#### 3. Add to AdGuard Home
* Navigate to **Filters > DNS Blocklists**.
* Click **Add blocklist > Add a custom list**.
* Name it `F* Those Guys!` and paste the URL.
